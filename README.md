# Java Number Guessing Game

## Prerequisite

[Java SE](https://www.oracle.com/technetwork/java/javase/downloads/index.html)

## How to Launch

1. `git clone` this repository
1. `cd` into the clonned repository's directory
1. Run `java -jar NumberGuess.jar`
